Nakloid
======================
Nakloid: Unit-waveform-oriented Singing Voice Synthesis System

required
------
* [Boost C++ Libraries](http://www.boost.org/)
* [FFTSS](http://www.ssisc.org/fftss/index.ja.html)
* [RapidJSON](https://github.com/miloyip/rapidjson/)
